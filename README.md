# 📝 SwiftUI To-Do App

A basic to-do list app built with SwiftUI for learning purposes.

## 🔧 Features
- Add, delete, and complete tasks
- Uses @State and List
- Simple clean interface

## 📚 SwiftUI Concepts
- @State management
- List & ForEach
- View composition
- TextField input
- onDelete (swipe to delete)

## 🚀 Next Steps
- Local persistence with UserDefaults
- Due dates and categories
- Dark mode support
